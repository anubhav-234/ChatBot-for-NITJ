About NITJ Chatbot

This is the machine learning chatbot application for users convinience inorder to search the information available on the NITJ website.

Chatbot is developed including following fetures :-

  • Machine Learning Model is implemented in the python using flask, natural lanaguage processing(nltk) modules, flask, pyTorch etc.
  • There is a files named 'intents.json which stores the dataset required for its model traning. See Sample json file here...
  • A Rest API Server is developed named 'app.py' which implements namely two Api's
    • API to render html component :- '/' is endpoint for it.
    • API to fetch the response from server when someone sends the text message to the server :- '/predict' is endpoint for it.
  • One of the important parameter which is the threshold for the category selection for the response type is kept as 0.70 for the shake of limited dataset present.
